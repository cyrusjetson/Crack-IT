# Bitstuffing

Domain : Computer Networks and Security
Topic : Bit Stuffing

Given a series of binary data, stuff the data as per the below procedures.

if there is a sequence as '011111' stuff the data with a zero.

Eg : Data - 01111111 => 011111011
     Data - 01111    => 01111
     
Destuffing is the reverse of Stuffing
